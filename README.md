## Details

**Features:**
* Register
* Login
* Transfer
* Deposit
* Withdraw
* Transaction History
* Filter,sort & search List items

## Backed & API

* The source code for the Api server can be found  [here](https://github.com/shittu33/HerokuNewsApiServer)
* The backend is built with PHP
* The Api use Slim Framework
